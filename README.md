### Options

| Option | Type | Default Value | Description | Range |
| :---: | :---: | :---: | :---: | :---: |
| native | Boolean | false | Load native emoji instead of image. |  N/A |

#### Option Example:

```vue
<EmojiPicker 
    :native="true" 
/>
```
